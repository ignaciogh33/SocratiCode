El presente Trabajo de Fin de Grado ha logrado desarrollar SocratiCode, un tutor socrático de programación que opera íntegramente en local y cumple con los objetivos iniciales de privacidad y autonomía del estudiante. Se ha implementado un sistema robusto que combina un \textit{system prompt} estructurado en capas que condiciona el comportamiento pedagógico del LLM sin reentrenamiento, un flujo de respuesta en tiempo real basado en Server Sent Events, un sistema de moderación dual que protege el entorno educativo y un motor de ejecución de código aislado mediante contenedores efímeros. La aplicación ofrece una interfaz de página única que integra editor, terminal y chat en un único espacio visual, garantizando el funcionamiento sin dependencia de APIs comerciales ni envío de datos a terceros.
