SocratiCode es una aplicación web de carácter educativo diseñada para asistir a los estudiantes de programación mediante tutoría interactiva. Su objetivo principal es facilitar el aprendizaje autodidacta del código a través del método socrático, guiando al usuario hacia las soluciones en lugar de proporcionárselas directamente. Frente a las herramientas de inteligencia artificial comerciales basadas en la nube, este sistema adopta un enfoque pedagógico basado en el diálogo socrático y la inferencia local. De esta forma, se evita la dependencia de infraestructuras de terceros, se garantiza la privacidad de los datos académicos y se asegura que la IA actúe como un facilitador del pensamiento, no como un sustituto del esfuerzo cognitivo.

Esta plataforma brinda a los alumnos un espacio donde puedan utilizar la potencia de la inteligencia artificial aplicada a la programación, sin que esta sustituya su proceso de razonamiento ni les prive de desarrollar un pensamiento computacional propio. Para alcanzar esta meta, SocratiCode implementa un conjunto de funcionalidades diseñadas para integrarse de manera natural en la rutina de estudio del usuario. Sus características principales incluyen un chat socrático interactivo que supervisa las respuestas en tiempo real para evitar la entrega directa de código, un editor de código con resaltado de sintaxis junto a una consola de resultados integrada que muestra la salida del programa y trazas de error, un sistema de generación automática de títulos para organizar el historial de consultas y la gestión personalizada de perfiles de usuario así como la configuración de modelos para administradores. Toda esta funcionalidad se presenta a través de una experiencia de usuario (UX) inmersiva, limpia y \textit{responsive}, donde la navegación se centra en el área de conversación y elimina distracciones, emulando la estética de las herramientas de desarrollo modernas.

El sistema ha sido diseñado teniendo en cuenta las exigencias técnicas de integrar modelos autorregresivos en aplicaciones web, como la alta latencia computacional y la necesidad de mantener el estado de la sesión. Con el objetivo de lograr un equilibrio entre la complejidad del procesamiento de la IA, el rendimiento del sistema y la fluidez de uso, se ha adoptado una arquitectura de \textit{Single-Page Application} (SPA) combinada con el protocolo \textit{Server-Sent Events} (SSE). Esta configuración permite transmitir las respuestas del tutor fragmento a fragmento (\textit{streaming} de \textit{tokens}) y ofrecer una interactividad continua sin recargas de página que interrumpan el flujo de trabajo.

En conclusión, este proyecto busca proporcionar a estudiantes y docentes una alternativa práctica, segura y metodológicamente robusta para el aprendizaje introductorio de la programación, mitigando la frustración inicial mediante asistencia guiada en tiempo real, al mismo tiempo que se fomenta la autonomía del usuario en un entorno completamente autocontenido.
