Este apartado aborda la validación externa de SocratiCode mediante una encuesta de evaluación que mide la usabilidad y la efectividad pedagógica del método socrático. Los participantes realizaron tres tipos de ejercicios: desarrollo de un script en Python con diccionarios, corrección de errores lógicos en código preexistente y simulación de escenarios adversarios en los que intentaron forzar al modelo a revelar la solución directa o extraer respuestas ajenas al ámbito de la programación.

Para llevar a cabo esta evaluación, se desarrolló una versión de la aplicación web que usaba la API de \textbf{OpenRouter}\footnote{\url{https://openrouter.ai/}} para facilitar el acceso a modelos que serían totalmente viables de ejecutar de forma local para una institución, pero que resultaban inasumibles por limitaciones propias de hardware.

Para realizar la encuesta se distinguió entre tres perfiles de usuarios, personas sin formación en ingeniería, ingenieros de otras especialidades no centrados en el desarrollo de software y estudiantes de Ingeniería Informática con conocimientos específicos en la materia.

Los resultados de los ejercicios prácticos confirman el éxito del modelo pedagógico y de seguridad. La inteligencia artificial se abstuvo sistemáticamente de proporcionar el código final, priorizando la guía deductiva del alumno. Asimismo, el sistema demostró una solidez absoluta frente a los intentos de manipulación de contexto (\textit{jailbreak}), manteniendo intacto su rol como tutor de programación en todos los escenarios evaluados.
