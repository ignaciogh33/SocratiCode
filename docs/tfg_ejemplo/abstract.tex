Learning to program is one of the greatest challenges in current technological education. Students often get stuck on syntax or logic errors, and conventional generative AI tools (such as ChatGPT or GitHub Copilot) tend to provide complete solutions immediately. This encourages \textit{copy-pasting} and reduces meaningful learning.

This Bachelor's Thesis presents the design and implementation of a Socratic Tutor based on Large Language Models (LLMs). Through advanced \textit{prompt engineering} techniques, the system acts as a pedagogical guide that refuses to deliver complete functional code, opting instead to ask reflective questions, point out clues, and help the student reach the solution on their own.

The platform has been developed as a complete web application, integrating an interactive frontend and a robust backend capable of managing conversation history and session context. Tests show that users prefer this interaction for deep learning compared to conventional autocomplete assistants.
